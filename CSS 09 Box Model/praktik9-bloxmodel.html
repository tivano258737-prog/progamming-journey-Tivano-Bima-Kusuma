Blog Sederhana

Contoh blog sederhana dengan blog model

Box model adalah model yang dipakai untuk menampilkan element HTML di browser. Model ini akan memudahkan kita mengatur ukuran, posisi, dan properti lainnya.

Dengan memahami model, kita akan mudah mengatur space pada element. Ini berguna dalam membuat layout supaya terlihat lebih rapi dan estetik.
